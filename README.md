# 📖 Overview

This project performs a comprehensive **Exploratory Data Analysis (EDA)** on a Billboard music dataset containing information about artists, songs, genres, moods, tempos, and chart performance.

The objective is to transform raw music industry data into meaningful insights by applying data cleaning, statistical analysis, feature engineering, and visualization techniques.

The analysis simulates how a music streaming platform or record label could leverage historical chart data to better understand artist demographics, music trends, and audience preferences.

---

# 🎯 Business Problem

Music companies generate massive amounts of data related to artists and song performance.

Understanding relationships between genres, artist characteristics, moods, and chart success can support decisions in:

- Marketing campaigns
- Artist promotion
- Playlist curation
- Audience segmentation
- Music trend analysis

This project demonstrates how exploratory analytics can uncover valuable business insights from historical music data.

The dataset contains approximately **635 Billboard chart records**, including:

- Song Title
- Artist
- Genre
- Mood
- Tempo
- Artist Type
- Billboard Chart Date
- Song Duration
- Top Chart Indicator
- Artist Birth Year
